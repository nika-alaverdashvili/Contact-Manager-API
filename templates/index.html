Contact Manager Flask API

This is a sample Flask application that serves as a web service, allowing you to perform CRUD operations (Create, Read, Update, Delete) using various HTTP methods. The available methods are POST, GET, PUT, and DELETE.

With this web service, you can send POST requests to create new information, GET requests to retrieve existing information, PUT requests to update information, and DELETE requests to remove information.
